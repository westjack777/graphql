# requirements

- Apollo Server
- GraphQL
- JSON Server
- Node Fetch
- Python

```shell
$  npm install apollo-server graphql json-server node-fetch python
$  npm install sucrase nodemon -D
```

## start servers

- Used to start the Apollo Server and the JSON Server

```shell
$  npm run dev
$  npm run server
```
